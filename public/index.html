About Cadence

Cadence is an online radio with a heavy focus on anime and Asian music.

This project was originally started in February 2017 as my first endeavour to practice a full range of tech skills, from development to server administration. The current iteration (4.x) of Cadence is written in Go without any frameworks. Cadence provides users with the ability to search the library as well as make song requests.

Active development continues to this day. All source is available on GitHub.

Cadence Radio is a DMCA compliant, "non-commercial educational" (NCE) broadcast. As an NCE, Cadence Radio is non-profit and does not accept advertisements for its webpages or broadcasts.
