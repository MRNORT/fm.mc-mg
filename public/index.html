About Cadence

Cadence is an online radio originally started in February 2017 as my first endeavour to practice a full range of tech skills, from development to server administration.

The application is a Go web server which interacts with an Icecast stream server and database server. Cadence provides users with the ability to search the library as well as make song requests.

Active development continues to this day. All source is available on GitHub.

Cadence Radio is a DMCA compliant, "non-commercial educational" (NCE) broadcast. As an NCE, Cadence Radio is non-profit and does not accept advertisements for its webpages or broadcasts.
