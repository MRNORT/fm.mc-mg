About Cadence:

Cadence is an anime-inspired online radio. Originally started as an experiment in February 2017, the project is a result of my first endeavour to practice a full range of IT-skills in web server programming, front-end design, Linux administration, databases, networking, and cybersecurity.

Cadence is essentially a web application which interacts with an Icecast stream server. Cadence further provides users with the ability to search through a music database as well as make requests.

The music library is diverse, with songs inclusive of almost every genre. I occasionally override the server and play a certain genre, artist, or the same song on an endless loop.

Github: Code/Issues. Diagram: Application Architecture

Cadence Radio is a DMCA compliant, "non-commercial educational" (NCE) broadcast. As an NCE, Cadence Radio is non-profit and does not accept advertisements for its webpages or broadcasts.

Announcements

02/01/2019: New Theme - Psycho Pass

This is the first major theme update in a long time. Psycho-Pass was one of the first themes I wanted to do but just never got around to. In comparison to Cyberpunk Bartender, The Psycho-Pass theme is a truer, darker, more dystopian cyberpunk theme. It's heavier in detailing than most themes I've done before. I'm extremely pleased with the way it turned out.

As a special event for the new theme, I'll be overriding the server to play from the Psycho-Pass OST. Songs from Cadence's regular programming will still be available for request through ARIA.

12/26: iOS/MacOS Stream Issue

A known issue is preventing the Cadence stream from playing across iOS devices, as well as the Safari browser on MacOS. If I still had the old issue tagging system, this issue would be "IMPORTANT".

At the current time, I have no workaround for iOS users. This issue seems to affect all iOS browsers. If you are using a Mac, the issue seems limited to Safari only, and you can still access the stream on Chrome. I will be working on a permanent fix in the meantime.
